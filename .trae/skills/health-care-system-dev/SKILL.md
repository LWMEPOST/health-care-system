# Health Care System Developer

This skill is designed to assist with the development of the Health Care System project.

## Project Overview
- **Frontend**: Vue 3 + TypeScript + Vite + Element Plus + Pinia
- **Backend**: Supabase (PostgreSQL, Auth, Storage)
- **AI**: DeepSeek API integration

## Key Responsibilities
- Implementing Vue 3 components and views.
- Managing state with Pinia.
- Interacting with Supabase for authentication and data storage.
- integrating AI features using DeepSeek API.

## Development Guidelines
- Follow the project structure: `src/components`, `src/views`, `src/store`, `src/services`.
- Use TypeScript for type safety.
- Ensure all new features are responsive and follow the design system.
- Use `Element Plus` for UI components.

## Common Tasks
- **Authentication**: Manage user login, registration, and session handling via Supabase.
- **Health Data**: Implement CRUD operations for health metrics.
- **Recipes**: Manage recipe data and recommendations.
- **AI Integration**: Use `DeepSeekService` for generating health advice.

When working on this project, always refer to the existing codebase and follow the established patterns.
